# Telegram Bot Skeleton: оплаты + доступ в канал/группу

Этот репозиторий — **пустая структура** (скелет) под бота:
- меню `/start` → “Доступы” / “Связь с менеджером”
- выбор тарифа → подтверждение → выбор способа оплаты (CryptoBot / CactusPay)
- создание заказа, оплата, авто-отмена при таймауте
- выдача инвайта и управление сроком доступа (неделя/месяц/пробник)
- уведомления админу

**Технологии (план):**
- Python 3.11+
- aiogram v3 (асинхронный Telegram Bot API)
- SQLite + aiosqlite (по умолчанию) / можно заменить на Postgres
- планировщик на asyncio (проверка оплат / истечения подписок)

## Быстрый старт (macOS)
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
python -m app.app
```

## Важно про права бота
Бот должен быть **админом** в приватной группе/канале:
- создавать инвайт-ссылки
- кикать (ban/unban) пользователей по окончании подписки

## Где лежат картинки
Все картинки — в `assets/images/`.
